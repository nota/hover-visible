Hover Visible Demo

What to check:

  • When using a mouse, the hover style is shown.
  • When using a pen or touch, the hover style is not shown.
    • The default :hover behaivior on android is, if you tap the element, the hover style remains showing.
  • When any key (except for modifier key) is pressed, the hover style will be hidden.
